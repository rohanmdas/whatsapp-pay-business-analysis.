# Payment Process Flow

This section illustrates the operational workflow for WhatsApp Pay including the current system state and the proposed payment workflow.

## Current State & Future State

The current WhatsApp platform primarily supports messaging, media sharing, and communication features.
The future state introduces an integrated payment feature enabling users to send and receive money directly through WhatsApp using the UPI payment infrastructure.

## System Context

The payment system interacts with several external entities including banks, UPI infrastructure, and regulatory systems.
