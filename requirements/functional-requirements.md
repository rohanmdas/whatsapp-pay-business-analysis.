# Functional Requirements

Functional requirements describe the core capabilities of the WhatsApp Pay system.

## User Account Setup

- User registration and login
- Bank account linking through mobile number verification
- UPI ID creation

## Payment Features

- Send money to WhatsApp contacts
- Receive payment requests
- QR code payment support
- Payment confirmation notifications

## Transaction Management

- Transaction history
- Payment status tracking
- Ability to remove bank account or deactivate payment service
